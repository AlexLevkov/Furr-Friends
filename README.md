## FurrFriends

FurrFriends is a fullstack pet adoption app.<br/>
[Run app 🚀](https://furr-friends.herokuapp.com/#/)

### 🖥️ Main Features:
* Adoption requests CRUD
* Search engine 
* Search for random pet
* User's profile page
* Pet's profile page
* Share on facebook
* Compitability to mobile and desktop

### ⚙Tech Stack :

#### Frontend:
* Vue.js
* Vuex 
* Vue router
* Axios 
* Chart.js 
* Element ui 
* Sass
* Socket.io 

#### Backend:
* Node.js 
* MongoDB
* Express 
* Express-session 
* Socket.io 
* Bycrypt 
* Cors
